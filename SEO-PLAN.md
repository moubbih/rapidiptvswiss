## 3. Technical SEO Roadmap

### Month 1 (April) -- Foundation
- [ ] Add hreflang tags linking EN/DE/FR blog post equivalents
- [ ] Implement JSON-LD schema: Organization, Product, FAQ, HowTo
- [ ] Add structured breadcrumbs with BreadcrumbList schema
- [ ] Audit and fix meta descriptions (unique per page, 150-160 chars)
- [ ] Compress images to WebP, add explicit width/height to prevent CLS
- [ ] Submit sitemap.xml to Google Search Console and Bing Webmaster Tools

### Month 2 (May) -- Performance
- [ ] Target Core Web Vitals: LCP < 2.5s, CLS < 0.1, INP < 200ms
- [ ] Implement lazy loading for below-fold images (channel logos)
- [ ] Add canonical URLs to all pages
- [ ] Create XML sitemap index (separate sitemaps for pages, blog, guides)
- [ ] Internal linking audit: every blog post links to 2-3 related posts + /plans

### Month 3 (June) -- Expansion
- [ ] Build Italian language section (/it/) with 5 core pages
- [ ] Add review/testimonial schema (AggregateRating)
- [ ] Implement dynamic OG images for social sharing
- [ ] Set up Google Business Profile for local presence
